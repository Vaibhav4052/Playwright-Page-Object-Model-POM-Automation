# Playwright Page Object Model (POM) Automation Framework

## Overview
This project demonstrates Page Object Model (POM) based UI automation for a B2B SaaS web application.

## Skills Covered
- Page Object Model (POM)
- Playwright with Pytest
- Maintainable UI automation
- CI/CD ready structure

## Framework Structure
pages/
  login_page.py
  dashboard_page.py
tests/
  test_login_flow.py
utils/
  base_page.py

## Why POM?
POM improves test readability, reusability, and maintenance by separating UI logic from test logic.

## Ideal For
Resume shortlisting and automation interviews.
